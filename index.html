This project harnesses Computer Vision (CV) technology to monitor construction workers and ensure they're wearing their safety gear, such as hard hats and vests. By using advanced image processing algorithms and machine learning models like YOLOv3, it's like having a smart system that acts as an extra set of eyes on the construction site. If someone forgets their gear or doesn't wear it properly, the system alerts the supervisor in real-time. This helps maintain safety without the need for constant manual supervision.
